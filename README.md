# faker-js

## Usage
Simple copy of faker.js library prepared as project at University - generate massive amounts of fake data in the browser and node.js
###

## Example
```javascript
const faker = require("@playerony/faker-js");

console.log(faker.date.future());
```

###

## API methods
date:
- past
- future
- recent
phone:
- phoneNumber
name:
- firstName
- lastName
- fullName
internet:
- email
- mac
- ipv4
- ipv6
type:
- number
- string
- boolean
